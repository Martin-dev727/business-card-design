# Business Card Design

This project is a simple business card design built using HTML and CSS.

## Features
- Clean and simple layout  
- Center-aligned content  
- Responsive profile image  
- Basic styling with CSS  

## Technologies Used
- HTML
- CSS

## Purpose
This project is part of my learning journey in web development.  
It helps practice layout, spacing, typography, and basic CSS properties.

## How to Use
1. Clone or download the repository  
2. Open `index.html` in your browser  
3. Edit the content and styles to customize the card
